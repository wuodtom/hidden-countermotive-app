# Hidden Countermotive Interpreter v2

A Streamlit app that models:

Observed phenomenon -> stated surface function -> inferred hidden driver

## New in v2

- text auto-scoring from user input
- CSV export
- PDF report export
- domain presets
- confidence score
- blended manual + auto feature scoring

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
